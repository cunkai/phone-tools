# Android Tools

将以下工具放入此目录，打包时会自动包含：

## Windows
- `adb.exe` - Android Debug Bridge
- `AdbWinApi.dll` - ADB Windows API
- `AdbWinUsbApi.dll` - ADB Windows USB API
- `aapt2.exe` - Android Asset Packaging Tool 2

## macOS / Linux
- `adb` - Android Debug Bridge
- `aapt2` - Android Asset Packaging Tool 2

## 获取方式

1. 安装 Android Studio，从 SDK Manager 中下载
2. 或直接下载 [Android Command Line Tools](https://developer.android.com/studio#command-tools)
3. adb 位于 `platform-tools/` 目录
4. aapt2 位于 `build-tools/<version>/` 目录
